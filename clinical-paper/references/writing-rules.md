# Academic Writing Rules — 13ルール + AI表現回避

全セクションのドラフトに適用する品質ルール。

---

## 13 Core Rules

### 1. 用語統一 (Terminology consistency)
- 1つの概念に対する英語表現は1つに固定する
- 初出で定義した略語・大文字小文字を以後も厳密に踏襲する

### 2. 過剰主張の抑制 (Claim strength calibration)
- RCT以外では因果を断定しない
- 観察研究では "was associated with" 等の関連表現を基本とする
- "prove/definitely/breakthrough" 等の強い断定語は避ける

### 3. 具体性 (Specificity)
- 曖昧語（many/several/a lot）を避ける
- 可能なら数・割合・効果量（95% CI等）で具体化する（数値が不明なら不明と明示）

### 4. 明瞭な照応 (Clear reference)
- this/these は曖昧に使わず、"this finding/this association/this result" のように参照対象を名詞で明示する

### 5. 簡潔性 (Conciseness)
- in order to → to
- due to the fact that → because
- a total of N patients → N patients
- it is worth noting that → 削除

### 6. 名詞化の抑制 (Limit nominalization)
- 名詞の連結で重くしない。可能な限り動詞で書く
- 3語超の名詞チェーンは再構成する

### 7. 主語—動詞の近接 (Subject-verb proximity)
- 主語と動詞の間が10語超なら後置修飾へ移すか2文に分割する

### 8. 能動態/受動態の一貫性 (Voice consistency)
- Methods: 受動態が自然な場面もあるが、乱用は避ける
- Results/Discussion: 能動態を優先
- 同一段落内で混在させすぎない

### 9. 接続の機能選択 (Connector discipline)
- 追加: In addition, Moreover（控えめに）, Also
- 対比: However, In contrast, While, Whereas, Although
- 因果: Therefore, Thus, Consequently
- 例示: For example, For instance, Specifically
- 要約: In summary, Overall, Taken together
- "However" は1セクション2回まで。構造的対比（While X, Y…）を優先する
- 連続する段落を同じ接続詞で始めない

### 10. 1文1アイデア (One idea per sentence)
- 1文に結果と解釈を詰め込まない
- 2つの独立した知見は分割する

### 11. 文長の目安 (Sentence length)
- 平均15-25語を目標
- 最大約35語。超えたら分割
- 従属節は原則1つまで

### 12. 段落単位の論理 (Paragraph structure)
- Topic sentence（主張） → Evidence（根拠） → Interpretation（必要時のみ） → Closing/Transition（橋渡し）
- 1段落1メッセージ

### 13. セクション間の整合 (Section coherence)
- Introduction: 確立知識は現在形、特定の先行研究は過去形
- Methods: 全体を過去形
- Results: 所見は過去形
- Discussion: 一般的主張は現在形、自分の結果は過去形

## AI-Phrasing 回避リスト

| 避ける表現 | 代替表現 |
|---|---|
| utilize | use |
| demonstrate | show |
| elucidate | clarify, explain |
| facilitate | help, enable |
| comprehensive | thorough, full |
| robust | strong, reliable |
| leverage | use, apply |
| underscore | emphasize, highlight |
| pivotal | key, central |
| delve into | examine, explore |
| cutting-edge | advanced, latest |
| groundbreaking | novel, new |
| it is important to note | （削除） |
| it should be noted that | （削除または書き換え） |
| in recent years | （具体的に: "Since 2020," 等） |
| a growing body of evidence | （具体的な研究を引用） |

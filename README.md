# Claude Skills ライブラリ

Claude用Skillsの配布リポジトリです。

## Skills一覧

| Skill名 | 概要 |
|---|---|
| [app-guide](./app-guide/) | 完全初心者向け取説を自動生成するSkill |
| [humanize](./humanize/) | AIが書いた日本語を自然な日本語にリライトするSkill |
| [graphic-record](./graphic-record/) | 臨床論文のグラフィックレコードを作成するSkill |
| [academic-write](./academic-write/) | 英語医学論文の執筆・校正支援Skill |
| [clinical-paper](./clinical-paper/) | 医学論文のセクション別執筆支援Skill |
| [pubmed-systematic-review](./pubmed-systematic-review/) | PubMedから文献を系統的に検索・抽出・整理するSkill |
| [case-report](./case-report/) | 英文症例報告の執筆支援Skill |

## ダウンロード方法

右上の緑色の「Code」ボタン →「Download ZIP」で全Skillsをまとめてダウンロードできます。
個別にダウンロードしたい場合は、各フォルダを開いて同様の操作をしてください。
